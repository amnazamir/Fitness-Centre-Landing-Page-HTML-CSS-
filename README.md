 💪 Fitness Centre Management System (Frontend)

This is a simple **Fitness/Gym website landing page** built using **HTML and CSS**.  
It is designed as a practice project for frontend development.



 📌 Features

- Responsive header with logo  
- Navigation menu  
- Gym themed background design  
- Registration form (Name, Age, Gender, Contact, Email)  
- Styled buttons and input fields  
- Card-based layout design  

---
 🛠️ Technologies Used

- HTML5  
- CSS3
